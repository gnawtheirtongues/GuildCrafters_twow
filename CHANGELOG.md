# Changelog

All notable changes to this project are documented in this file.

## 1.3.0

Improvements:

- Added predefined recipe categories across professions for consistent grouping.
- Added collapsible category headers in the recipe list.
- Added `Collapse/Expand` control next to `Refresh`.
- Category headers now render in white text for better readability.
- Category display now sanitizes bad source categories (for example `Mail`, `Plate`, `Staves`) into stable buckets.
- Improved recipe preview green text handling to use item tooltip "Use/Equip" lines where available.
- Kept Enchanting behavior intact while applying category and UI improvements to other professions.

## 1.2.0

Improvements:

- Fixed cooking exclusion system.
- Added profession exclusion handling.
- Improved catalog index filtering.
- Fixed incorrect profession assignments.

## 1.1.0

Improvements:

- Fixed engineering scan issues.
- Fixed recipe normalization.
- Added export rewrite command.
- Improved sync stability.

## 1.0.0

Initial public release.

Features:

- Guild crafting database.
- Profession scanning.
- Recipe sharing.
- Recipe search.
- Export database support.
